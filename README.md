# Space shooter LibGDX

## Description
Space shooter game developed by LibGDX framework.

## Tools and technologies used in project
- Java Core
- LibGDX framework
- Gradle

## Tools needed for launching project
- Java 8
- IntelliJ IDEA

## Instruction for launching project
- download project from GitHub
- open project folder by IntelliJ IDEA
- run desktop/src/com/star/app/desktop/DesktopLauncher.java

## Controls
- w – move forward
- s – move back
- d – turn right
- a – turn left
- space – shoot
- p – open shop
- r – launch rocket

## Описание
Игра – космический шутер, разработанная с помощью фреймворка LibGDX.

- w – двигаться вперед
